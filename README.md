## 📚 How to Use
Each configuration folder in this repository includes its own **README.md** with clear setup instructions.  
You can read them one by one and apply them directly to your system.

- `hypr/` → Hyprland configuration  
- `waybar/` → Custom Waybar config (with wallpaper picker)  
- `nvim/` or `.vimrc` → Vim/Neovim configuration  
- `cava/` → Cava (audio visualizer) configuration
